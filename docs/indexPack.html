indexPack

Create the .idx file for a given .pack file

paramtype [= default]description
fs, dir, gitdirFSModule, string, stringThe filesystem holding the git repo, the working tree directory path, and optionally the git directory path
filepathstringThe path to the .pack file to index.
returnPromise<void>Resolves when filesystem operations are complete
 
let repo = {fs, dir: '$input((.))'}
await git.indexPack({...repo, filepath: '$input((pack-9cbd243a1caa4cb4bef976062434a958d82721a9.pack))'})
console.log('done')
